# Cluster Resource Management & Scheduling Simulator

Данный проект представляет собой программный комплекс для моделирования и анализа алгоритмов размещения контейнеров на узлах кластера в условиях ограниченных ресурсов (CPU и RAM). Разработано в рамках выпускной квалификационной работы.

## Функциональные возможности
* **Визуализация размещения:** Интерактивное отображение заполненности узлов кластера.
* **Поддержка алгоритмов:**
  * `First Fit (z=1)` — размещение в первый подходящий узел.
  * `Best Fit (z=2)` — выбор узла с минимальным остатком ресурсов.
  * `Worst Fit (z=3)` — распределение нагрузки (выбор самого свободного узла).
  * `Genetic Algorithm (z=4)` — оптимизация размещения методом эволюционного моделирования.
* **Расчет метрик эффективности:**
  * Количество задействованных узлов ($N_{used}$).
  * Процент неразмещенных контейнеров ($P_{unplaced}$).
  * Средняя загрузка ресурсов кластера.
  * Дисперсия распределения нагрузки ($D$).
* **Гибкая настройка:** Возможность случайной генерации нагрузок или ручного ввода спецификаций через интерфейс.

## Стек технологий
* **Backend:** Python 3.12, Flask.
* **Математические вычисления:** NumPy.
* **Frontend:** HTML5, CSS3 (Modern UI/Dashboard), JavaScript (Vanilla).
